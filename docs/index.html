Cutting Edge Capabilities

Going beyond the basic building blocks of HTML, CSS, and JavaScript, the DEG UI team is well equipped to deliver on a number of cutting edge technologies. Have an idea for something, but don't see it? Don't hesitate to reach out! We're always eager to dive into new challenges and expand on our capabilities.

  • Progressive Web Apps

    PWA's aren't just a buzzword, they're where the web is heading. We have the capability to build reliable, fast, responsive, and installable applications out of any website.

  • Pattern Libraries

    Gone are the days of a static style guide being enough to fully represent a brand. We can incorporate Pattern Lab into any web project to automatically generate an interactive style guide and pattern library.

  • Mobile First Email Development

    DEG's UI team was one of the first in the world to develop in the responsive email space. Now, we've taken it a step further and created a methodology for truly mobile first email development that renders perfectly across any device or client.

  • Rapid Prototyping

    Need something built to quickly test out an idea? Or a proof of concept for a client? We have both the toolset and the ability to do just that without the need to spin up complex platforms or environments.

  • Experiential Technology

    There's more to creating an immersive, technology driven experience than just a pretty website. The DEG UI team excels at marrying cutting edge web technologies with crowd-pleasing real-world experiences.

  • Web AR & VR

    The immersive web means virtual world experiences are now available in the browser. And WebXR is the device API to get us there. Whether it's an AR or VR experience, the DEG UI team can help out with implementing this cutting edge technology.

  • Web Payments

    The way people pay online is rapidly changing. As sites move away from platform based checkouts and towards browser based Payment APIs, the DEG UI team is your go to for knowledge and implementation of these features.

  • Push Notifications

    Whether related to shipping orders, a user's location, or any number of other use cases, notifications are a valuable piece of any modern website or application. The DEG UI team is well versed in the Push and Notifications APIs, which brings this functionality to the web, whether the customer has their browser open or not.

  • Voice Apps & Interfaces

    Whether it's Alexa or in the browser, utilizing intents and utterances or the Web Speech API, the UI team has the background and experience to get custom voice apps and interfaces off the ground and running.

  • Performance Audits & Improvement

    In the pursuit to push the web to do more, we're running into a common problem: performance. Fortunately, the DEG UI team has the tools to not only evaluate performance but the skills to achieve a high level of performance across a variety of network conditions and devices.

  • Inclusive & Accessible Development

    We view accessibility as not only a legal issue, but as a moral and social imperative as well. To that end, we strive to build all solutions from websites to emails in the most accessible manner possible, adhering to WCAG 2.0 AA standards.

  • Device Sensors

    The introduction of Generic Sensor APIs opened up a literal world of new interaction capabilities. From Accelerometers to Gryoscopes to Ambient Light sensors, the DEG UI team is well versed in the technologies that allow us to build interfaces based on how users are interacting with their device and environment.

  • Interactive Email

    Emails no longer need to be static. From hotspots and animation to full-on in inbox ratings & reviews, whatever the interactive experience use case you can dream up, we've got the skills to build it.

  • And More...

    These are just a few examples of the cutting edge capabilities the DEG UI team is able to provide, but as we know well, the web and the technologies around it change daily. Have an idea or want us to help come up with one? We're always happy to help brainstorm ways to drive things forward.
